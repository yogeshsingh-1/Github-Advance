<!-- MongoDB Shell (mongosh) -->

- CLI (Command Line Interface) tool.
- Used to connect to and interact with a MongoDB database.
- Allows you to execute MongoDB commands and JavaScript.

<!-- MongoDB Database Server (mongod) -->

- The MongoDB database server.
- Stores, manages, and processes data.
- Runs in the background and listens for client connections.

<!-- MongoDB Compass -->

- GUI (Graphical User Interface) tool.
- Used to visually explore and manage MongoDB databases.
- Provides an alternative to the command-line shell.

<!-- Check MongoDB Shell version -->
mongosh --version

<!-- Check MongoDB Database Server version -->
mongod --version

Summary

mongosh  → MongoDB Shell (CLI Client)
mongod   → MongoDB Database Server
Compass  → MongoDB GUI Client
