<!-- What is wireshark? -->

Wireshark ek packet analyzer (network protocol analyzer) hai.

Iska kaam hai network interface se guzarne wale packets ko capture karna aur unhe decode karke readable form me dikhana.

Jaise:

Ethernet Frame
IP Packet
TCP Segment
UDP Datagram
HTTP Message
DNS Query
TLS Handshake

Ye sab Wireshark automatically parse karke dikhata hai.
